# Exploratory Data Analysis Report

## Dataset
- Source: scikit-learn Iris dataset
- Task: Multi-class classification
- Samples: 150
- Features: 4 numeric features
- Target: label with 3 classes

## Features
- sepal length (cm)
- sepal width (cm)
- petal length (cm)
- petal width (cm)

## Notes
- The dataset is well-suited for demonstrating end-to-end MLOps because it is small, public, and reproducible.
- The preprocessing step uses standard scaling and a train/test split.
