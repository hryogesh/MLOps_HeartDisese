# Project Summary

## Objective
Build an end-to-end MLOps project using a public dataset and demonstrate production-oriented engineering practices.

## Dataset
Iris dataset from scikit-learn.

## Workflow
1. Load data
2. Preprocess features
3. Train classifier
4. Evaluate model
5. Serve predictions through FastAPI
6. Test with pytest
7. Package with Docker
8. Automate with GitHub Actions

## Academic Use
This structure is suitable for an M.Tech submission because it combines machine learning, software engineering, deployment, and DevOps practices.
