# Final Report

## 1. Introduction
This project builds an end-to-end MLOps workflow for predicting heart disease using clinical data.

## 2. Dataset and EDA
The Cleveland heart disease dataset is loaded from the attached archive and cleaned for model training.

## 3. Feature Engineering and Modeling
Feature preprocessing, feature selection, and hyperparameter tuning are included. Logistic Regression and Random Forest models are compared.

## 4. Experiment Tracking
MLflow is used to track experiment parameters and metrics.

## 5. CI/CD and Deployment
GitHub Actions and Docker are used to automate testing and containerization. Kubernetes deployment manifests are included.

## 6. Monitoring and Logging
Logging is implemented in the API and Prometheus/Grafana files are present for monitoring setup.

## 7. Conclusion
The project demonstrates a production-style MLOps workflow suitable for academic submission and portfolio presentation.
