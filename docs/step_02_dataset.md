# Step 2 — Dataset Collection and Understanding

## Overview
The attached archive contains the Cleveland heart disease dataset. It is suitable for a binary classification task in an MLOps workflow.

## Dataset Source
- Archive: ../Downloads/heart+disease.zip
- File used: processed.cleveland.data

## Target Variable
The target indicates the presence of heart disease. The original values 0 to 4 were consolidated into binary labels:
- 0 = no disease
- 1 = disease present

## Features
The dataset includes 13 clinical input features used for classification.
