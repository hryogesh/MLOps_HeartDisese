# Project Summary for Submission

## Title
End-to-End MLOps Project for Heart Disease Prediction

## Problem Statement
Heart disease is a major health concern, and early prediction can support timely medical assessment. This project develops a machine learning system that predicts whether a patient is likely to have heart disease based on clinical attributes.

## Objectives
- Build a reliable machine learning pipeline
- Apply data validation, cleaning, and preprocessing
- Train and evaluate a classification model
- Track experiments with MLflow
- Serve predictions through FastAPI
- Provide a user interface using Streamlit
- Package the solution using Docker and deployment manifests

## Technologies Used
Python, scikit-learn, FastAPI, Streamlit, MLflow, DVC, Docker, Kubernetes, GitHub Actions, Pytest

## Workflow
1. Load the Cleveland heart disease dataset from the attached archive.
2. Validate and preprocess the data.
3. Perform feature selection and hyperparameter tuning.
4. Train the logistic regression model.
5. Evaluate the model and save the artifact.
6. Serve predictions through an API.
7. Deploy the solution using container and orchestration files.

## Outcome
The project demonstrates an end-to-end MLOps workflow suitable for academic submission, portfolio presentation, and practical deployment discussions.
